Setup Instructions

See docs/getting-started.md. For more information on installing Larry:

  1. Select "download" for the installation files from GitHub. You will see green "Code" button that creates a zip file of the Larry OS Starter Kit that downloads to your laptop.
  2. Install Obsidian.md on your Laptop. It is a free app.
  3. Signup and use Claude Cowork to install Larry OS. Tell Claude to use the provided Larry-pka-plugin that automatically installs the package.
  4. The starter kit will add a file folder to your desktop which will become the knowledge and intelligence folder for Larry.
  5. Start using Larry (FYI. you can rename Larry to something you relate to)
How do I download the kit from GitHub?

If you land on the GitHub repository page (github.com/rickkam503/larry-pka-starter) and need to download the kit manually:

  1. Look for the green Code button in the top-right area of the file listing.
  2. Click it. A small menu opens.
  3. Click Download ZIP at the bottom of that menu.
  4. The starter-kit zip file downloads to your machine.

Most visitors won't see this — the Download Larry PKA Starter Kit button on the Download tab points directly at the ZIP file, so clicking it skips the GitHub page entirely.

What is Larry?

Your AI orchestrator. He routes work to a team of specialists (Nolan, Pax, Mira, Cleo, Theo, Wren, Knox, Felix, Hank, Sloane, Nell). He never does the work himself.

What is Obsidian.md?

Obsidian.md becomes the "operational brain" and long-term memory layer for your PKA Larry starter kit.

In your ecosystem, Larry is essentially the orchestrator / personality layer, while Obsidian acts as:

  • the knowledge vault
  • memory system
  • document repository
  • relationship graph
  • operational workspace
  • AI context engine
How does work flow through Larry?

Every request flows through Larry. He routes, the team executes, he synthesizes the result back to you.

  1. Owner Human
  2. Owner Inbox
  3. Larry Reviews / Categorizes
  4. Assigns to Appropriate Agent
  5. Agent Produces Output
  6. Team Inbox
  7. Larry Reviews + Synthesizes
  8. Final Recommendation to Owner
What is the function of the Owner's Inbox and Team Inbox?

Function of the Team Inbox

The Team Inbox is the primary intake location from the human owner.

This is where the owner drops:

  • PDFs
  • screenshots
  • contracts
  • voice memos
  • meeting notes
  • financial exports
  • ideas
  • emails
  • handwritten scans
  • research
  • operational requests

Think of it as: “Tasks and information waiting for Larry.”

Larry monitors this inbox and:

  • interprets intent
  • routes work
  • extracts context
  • creates tasks
  • triggers workflows
  • archives completed items

Function of the Owner's Inbox

The Owner's Inbox is the shared outbound collaboration space used by AI agents and operational workflows.

This is where agents place:

  • completed drafts
  • reports
  • summaries
  • strategy recommendations
  • generated assets
  • workflows
  • spreadsheets
  • meeting summaries
  • action plans

Think of it as: “Output ready for Larry and the human owner.”

Larry can:

  • review outputs
  • synthesize multiple agent contributions
  • prioritize next actions
  • escalate important decisions
  • package final deliverables

Finder Integration Tip (Very Important)

A powerful optimization is to pin both inbox folders directly into the macOS Finder sidebar.

Recommended pinned folders:

  • Owner Inbox
  • Team Inbox
  • Obsidian Vault
  • Active Projects
  • Archive

This makes the workflow frictionless. Instead of:

  • searching through folders
  • opening apps repeatedly
  • dragging across windows

You can simply:

  • drag files directly into the Team Inbox for Larry to process
  • drop completed outputs into the Owner's Inbox
  • quickly review AI-generated deliverables
  • move finalized files into project folders

This turns Finder into a lightweight operational command center.

How do I trigger the morning brief?

Say Larry, morning brief or Larry, full brief with world news. Use the Try saying card above to copy the exact phrasing.

How do I add a new team member?

Larry will automatically add new team members based on the work the human needs done.

How do I update my profile?

Say Larry, update my profile. He'll rewrite Library/user-profile.md based on your answers.

Customize your AI team

You can delete AI team members if you don't need them. Ask Larry to remove them from the team and hire members who better fit your situation.

Change the name from Larry to something else

You can change the name of your AI orchestrator by asking Larry to help you with that change. Pick a name that represents someone you want to have as your executive assistant.

Add a badge on the Chat screen

You can tell Larry to add a badge to the chat screen so you know Larry is working with you.
